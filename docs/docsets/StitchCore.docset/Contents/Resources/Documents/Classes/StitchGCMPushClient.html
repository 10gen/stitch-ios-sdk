StitchGCMPushClient

public class StitchGCMPushClient: PushClient

StitchGCMPushClient is the PushClient for GCM. It handles the logic of registering and deregistering with both GCM and Stitch.

It does not actively handle updates to the Instance ID when it is refreshed.
